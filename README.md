# The-Top-250-Movies_PowerBi
🎬 Power BI Dashboard Project – IMDb Top 250 Movies

In this project, I analyzed and visualized movie data using Power BI to create interactive dashboards and generate meaningful insights.

Key skills applied in this project:

✅ Data Collection: Working with the IMDb Top 250 Movies dataset downloaded from Kaggle
✅ Data Cleaning: Using Power Query for data transformation and handling missing values
✅ DAX Calculations: Creating measures and analytical calculations for deeper insights
✅ AI Tools: Leveraging AI to support data understanding and categorization
✅ Data Visualization: Designing interactive dashboards with custom formatting and tooltips

Tools & Technologies:
Power BI | Power Query | DAX | Data Visualization
